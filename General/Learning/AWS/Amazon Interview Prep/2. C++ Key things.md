


#### Find length of array in C++:

Raw:
```C++
int length = sizeof(arr) / sizeof(arr[0]);
```
 Library Vector:
```C++
#include <vector>
vector<int> vec = {1, 2, 3, 4, 5};
int length = vec.size();  // Returns 5
```

There is a array library as well but it has a fixed size and you have to predefine the size, vectors are dynamic arrays


#### Hashmaps:
Integer Keys and Value:
```C++
#include <unordered_map>
#include <iostream>
int main() {
    unordered_map<int, int> hashmap;
    hashmap[1] = 100;
    hashmap[2] = 200;
    cout << hashmap[1] << endl;  // Output: 100
}
```

String Keys:
```C++
#include <unordered_map>
#include <iostream>
#include <string>
int main() {
    unordered_map<string, int> age;
    age["Alice"] = 30;
    age["Bob"] = 25;
    cout << age["Alice"] << endl;  // Output: 30
}
```


#### Auto Loop:
```C++
#include <vector>
#include <iostream>
vector<int> nums = {1, 2, 3, 4};
for (auto num& : nums) {
    cout << num << " ";  // prints: 1 2 3 4
}
```
